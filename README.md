# 📖 React Project <a name="about-project"></a>

Tenzies is an engaging dice game crafted with React, offering players an enjoyable experience of rolling and holding dice in pursuit of specific combinations. Built with intuitive user interactions, Tenzies provides a fun and interactive way to challenge your luck and strategic decision-making skills

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

```
Git
VS Code
Node
```

### Setup

```
cd my-folder
git clone git@github.com/Muhammad0602/tenzies-game.git
cd tenzies-game
```

### Install

```
npm install
```

### Usage

```
npm run build
npm start
```
